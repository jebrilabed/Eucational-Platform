## 🌟 Overview | نظرة عامة

**Academic Educational Platform** is a specialized web application designed to support Engineering and IT students in their academic journey. The platform provides organized, high-quality educational content, enabling students to access lectures, track their progress, and navigate through courses based on their academic level and semester.

**منصة التعليم الأكاديمي** هي منصة تعليمية متخصصة تهدف إلى دعم طلاب الهندسة وتكنولوجيا المعلومات في رحلتهم الأكاديمية من خلال محتوى منظم وشامل، مما يسهل الوصول إلى المحاضرات ومتابعة التقدم الدراسي.

---

## ✨ Key Features | المميزات الرئيسية

- **🎯 Targeted Content**: Courses organized by faculty (IT, CS, Engineering), academic level (1-5), and semester.
- **📊 Progress Tracking**: Real-time visualization of course completion status.
- **🔍 Advanced Filtering**: Easily find courses using level and semester filters.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **🌍 RTL Support**: Native Arabic interface with Right-to-Left layout and Cairo typography.
- **🎥 Integrated Video Player**: Seamless YouTube integration for a smooth learning experience.

## 🛠 Tech Stack | التقنيات المستخدمة

- **Structure**: Semantic HTML5.
- **Styling**: Tailwind CSS v4 for modern, utility-first design.
- **Logic**: Vanilla JavaScript for dynamic rendering and state management.
- **Icons**: FontAwesome 7.1.0.
- **Typography**: Google Fonts (Cairo).

## 🚀 Getting Started | البدء بالعمل

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jebrilabed/Eucational-Platform.git
   ```

2. **Open the project**:
   Simply open `index.html` in any modern web browser. No build step is required as it uses the Tailwind Play CDN and Vanilla JS.
